Scratch Programming Course Curriculum

Course Overview
Description Introduction to coding through interactive, project-based learning with Scratch
Audience Beginner students ages 8-14
Duration 8 modules (16 Days) (24 hours total)
Learning Objectives
1 Develop computational thinking Problem-solving skills, logical reasoning, pattern recognition
2 Understand programming basics Visual block concepts, sequence, loops, conditionals
3 Create projects Games, interactive stories, animations, real-world applications
