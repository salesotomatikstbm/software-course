Bootstrap UI Quiz: Grid, Icons & Sliders

1. What's the correct class structure for Bootstrap's 12-column grid system?

container > row > col-*
row > container > col-*
col-* > row > container
container > col-* > row

Bootstrap grid structure:

<div class="container">
  <div class="row">
    <div class="col-md-6">Column 1</div>
    <div class="col-md-6">Column 2</div>
  </div>
</div>

2. How would you create a layout that shows 3 columns on desktop, 2 on tablet, and 1 on mobile?

col-lg-4 col-md-6 col-12
col-4 col-6 col-12
col-desktop-3 col-tablet-2 col-mobile-1
col-grid-3 col-grid-2 col-grid-1

Responsive column classes:

<div class="row">
  <div class="col-lg-4 col-md-6 col-12">Item</div>
  <div class="col-lg-4 col-md-6 col-12">Item</div>
  <div class="col-lg-4 col-md-6 col-12">Item</div>
</div>

3. What's the Bootstrap 5 way to add consistent spacing between grid columns?

gap-* utility classes on the row
margin utilities on each column
padding utilities on each column
spacer divs between columns

Using gap utilities:

<div class="row gap-3">
  <div class="col">Item</div>
  <div class="col">Item</div>
</div>

Available sizes: gap-1 (4px) to gap-5 (48px)

7. What's the essential structure for a Bootstrap Carousel?

carousel > inner > items + controls + indicators
slider > slides + navigation
carousel-container > slide-items
slideshow > slide + buttons

Basic carousel structure:

<div id="myCarousel" class="carousel slide">
  <div class="carousel-inner">
    <div class="carousel-item active">
      <img src="..." class="d-block w-100">
    </div>
    <div class="carousel-item">
      <img src="..." class="d-block w-100">
    </div>
  </div>
  <button class="carousel-control-prev" data-bs-target="#myCarousel">
    <span class="carousel-control-prev-icon"></span>
  </button>
  <button class="carousel-control-next" data-bs-target="#myCarousel">
    <span class="carousel-control-next-icon"></span>
  </button>
</div>

8. How would you make a carousel auto-rotate with 5-second intervals?

data-bs-interval="5000" on carousel div
auto-rotate="5" on carousel div
interval: 5000 in JavaScript
Add CSS animation with 5s duration

Carousel with auto-rotation:

<div id="autoCarousel" class="carousel slide" 
  data-bs-ride="carousel" data-bs-interval="5000">
  ...
</div>

Or via JavaScript initialization:

const myCarousel = new bootstrap.Carousel('#autoCarousel', {
  interval: 5000,
  ride: 'carousel'
})

9. How can you customize the appearance of Bootstrap's carousel indicators?

Override CSS variables or write custom CSS
Use data-bs-indicator-style attribute
Add modifier classes like indicators-rounded
You can't customize the default indicators

Customizing indicators with CSS variables:

.carousel-indicators {
  --bs-carousel-indicator-width: 20px;
  --bs-carousel-indicator-height: 3px;
  --bs-carousel-indicator-active-bg: #ff6b6b;
}

Or with custom CSS:

.carousel-indicators button {
  width: 15px;
  height: 15px;
  border-radius: 50%;
  margin: 0 5px;
}

10. How would you create a responsive grid of cards that becomes a carousel on mobile?

Use grid classes with carousel at breakpoint
Create two separate layouts and toggle visibility
Use CSS transform on mobile
Bootstrap doesn't support this pattern

Combining grid and carousel:

<!-- Desktop Grid -->
<div class="d-none d-md-block">
  <div class="row row-cols-3 g-4">
    <div class="col"><div class="card">...</div></div>
    <!-- more cards -->
  </div>
</div>

<!-- Mobile Carousel -->
<div class="d-md-none">
  <div id="mobileCarousel" class="carousel slide">
    <div class="carousel-inner">
      <div class="carousel-item active"><div class="card">...</div></div>
      <!-- more slides -->
    </div>
  </div>
</div>
