Bootstrap Basics Quiz

1. What is the primary advantage of using Bootstrap?

Rapid development with pre-built components
Better JavaScript performance
Automatic SEO optimization
Replacement for custom CSS

Bootstrap provides:

  • Responsive grid system
  • Pre-styled components (buttons, cards, etc.)
  • Utility classes for common styling
  • Cross-browser compatibility

2. What's the difference between .container and .container-fluid?

.container has max-widths at breakpoints, .container-fluid is always 100% width
.container is for fixed layouts, .container-fluid is for flexbox
.container has padding, .container-fluid doesn't
There is no difference
<div class="container">Fixed-width container</div>
<div class="container-fluid">Full-width container</div>

.container breakpoints:

  • 100% width below 576px
  • 540px (sm)
  • 720px (md)
  • 960px (lg)
  • 1140px (xl)
  • 1320px (xxl)
