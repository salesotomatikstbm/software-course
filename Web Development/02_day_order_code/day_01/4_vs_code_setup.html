How to Set Up Visual Studio Code (VS Code)

Step 1: Download and Install VS Code

  • Visit the official website: https://code.visualstudio.com/
  • Click the download button for your OS (Windows, macOS, or Linux).
  • Open the installer and follow the instructions to complete the installation.

Step 2: Install the Live Server Extension

  • Open VS Code.
  • Click on the Extensions icon in the sidebar (or press Ctrl+Shift+X).
  • Type Live Server in the search bar.
  • Find the extension by Ritwick Dey and click Install.
  • After installing, right-click your HTML file and choose "Open with Live Server".
This allows you to view your webpage live in the browser.

Step 3: Install Prettier (Code Formatter)

  • Go to the Extensions view again (Ctrl+Shift+X).
  • Search for Prettier - Code formatter.
  • Click Install on the extension by Prettier.
  • To enable auto-formatting: go to Settings → search "Format On Save" and enable it.
Prettier helps keep your code neat and consistent automatically.

Step 4: Ready to Code!

Now you're all set to write and preview HTML, CSS, and JavaScript code in real-time with auto-formatting support.

You’ve turned your computer into a mini coding lab!
