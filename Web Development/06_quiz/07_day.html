Responsive Design: Media Queries & Viewport Units

1. What is the primary purpose of CSS media queries?

To apply styles based on device characteristics
To add multimedia content to web pages
To optimize images for different devices
To create animations in CSS

Media queries allow you to create responsive designs:

@media (max-width: 768px) {
  /* Styles for mobile devices */
}

2. Which of these is NOT a viewport unit?

vh
vw
vmin
vpx

Viewport units:

  • vw - 1% of viewport width
  • vh - 1% of viewport height
  • vmin - 1% of viewport's smaller dimension
  • vmax - 1% of viewport's larger dimension

3. What is a "breakpoint" in responsive design?

A point where the website layout changes
A debugging tool in browsers
A CSS property for line breaks
A JavaScript error

Common breakpoints:

/* Mobile */
@media (max-width: 767px) {...}

/* Tablet */
@media (min-width: 768px) and (max-width: 1023px) {...}

/* Desktop */
@media (min-width: 1024px) {...}

4. What does "mobile-first" design approach mean?

Designing for mobile first, then enhancing for larger screens
Creating separate websites for mobile and desktop
Using only mobile devices for development
Making all websites look the same on every device

Mobile-first CSS structure:

/* Base styles (mobile) */
body { font-size: 14px; }

/* Tablet */
@media (min-width: 768px) {
  body { font-size: 16px; }
}

/* Desktop */
@media (min-width: 1024px) {
  body { font-size: 18px; }
}

5. What does 100vh represent?

100% of the viewport height
100% of the element's parent height
100 pixels in height
100% of the document height
This box is 50vh tall (50% of viewport height)

Viewport units are relative to the browser window size, not the document or parent element.

6. What is the correct syntax for a media query targeting screens between 600px and 900px wide?

@media (width >= 600px) and (width <= 900px)
@media (min-width: 600px) and (max-width: 900px)
@media screen (600px to 900px)
@media width(600-900px)
@media (min-width: 600px) and (max-width: 900px) {
  /* Styles for screens between 600px and 900px */
}

7. An element with width: 50vw will be how wide on a 1200px wide viewport?

600px
50px
1200px
2400px
This box is 50vw wide

1vw = 1% of viewport width, so 50vw = 50% of viewport width (600px on a 1200px viewport)

8. How would you target landscape orientation in a media query?

@media (orientation: landscape)
@media (landscape)
@media (rotate: 90deg)
@media (display: landscape)
@media (orientation: landscape) {
  /* Styles for landscape orientation */
}

This is useful for adjusting layouts when a device is rotated.

9. What does 10vmin represent on a viewport that's 1000px wide and 800px tall?

80px (10% of the height)
100px (10% of the width)
180px (10% of width + height)
10px (fixed size)

vmin is 1% of the viewport's smaller dimension:

  • Viewport: 1000px × 800px
  • Smaller dimension: 800px (height)
  • 10vmin = 10% of 800px = 80px

10. Which media feature would you use to detect a touchscreen device?

@media (hover: none)
@media (touch-enabled: true)
@media (pointer: coarse)
@media (screen: touch)

Modern approaches to detect touch devices:

/* Device has coarse pointer (like a finger) */
@media (pointer: coarse) {
  /* Touch device styles */
}

/* Device cannot hover (likely touch) */
@media (hover: none) {
  /* Touch device styles */
}
