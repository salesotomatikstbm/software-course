Website Structure

Just like a house has rooms and walls, a website has a structure made using HTML elements. These elements are like building blocks that create the layout of a page.

Main Parts of a Website:
  • <header> – Top section, usually has the logo and navigation
  • <nav> – Navigation links (menu)
  • <main> – Main content area (articles, info, images)
  • <section> – Divides the page into parts (like chapters)
  • <footer> – Bottom part (contact info, copyright)
Think of a website like a newspaper:
Header = Title of the paper
Nav = Table of contents
Main = Articles
Footer = Publisher and contact info
Example Structure:
<header>
    <h1>My Website</h1>
</header>

<nav>
    <a href="#home">Home</a>
    <a href="#about">About</a>
</nav>

<main>
    <section>
        <h2>Welcome!</h2>
        <p>Thanks for visiting my page.</p>
    </section>
</main>

<footer>
    <p>© 2025 My Website</p>
</footer>
