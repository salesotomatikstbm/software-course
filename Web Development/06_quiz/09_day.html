3. Where is the correct place to insert CSS in an HTML document?

CSS can be added in three ways:

/* External CSS */
<link rel="stylesheet" href="styles.css">

/* Internal CSS */
<style>
  body { background-color: white; }
</style>

/* Inline CSS */
<p style="color:red;">This is a paragraph.</p>

7. What is the purpose of CSS media queries?

To apply different styles based on device characteristics
To add multimedia content
To query a database
To create animations
@media (max-width: 768px) {
  /* Styles for screens smaller than 768px */
  body {
    font-size: 14px;
  }
}

8. Which media query targets screens with a maximum width of 600px?

@media (min-width: 600px)
@media screen and (max-width: 600px)
@media (device-width: 600px)
@media (width <= 600px)
@media screen and (max-width: 600px) {
  .menu {
    display: none;
  }
}

9. What is the purpose of the viewport meta tag?

To control the page's dimensions and scaling on mobile
To set the background image
To define the website's boundaries
To create a visible border around the page
<meta name="viewport" content="width=device-width, initial-scale=1.0">

This tag ensures proper rendering and touch zooming on mobile devices.

10. In mobile-first design, how should media queries typically be structured?

Start with mobile styles, then add larger screens with min-width
Start with desktop styles, then adjust for mobile with max-width
Create separate stylesheets for each device
Use only pixel values for all sizes
/* Mobile styles (default) */
body { font-size: 14px; }

/* Tablet styles */
@media (min-width: 768px) {
  body { font-size: 16px; }
}

/* Desktop styles */
@media (min-width: 1024px) {
  body { font-size: 18px; }
}
