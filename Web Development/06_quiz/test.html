3. How do you link a CSS file to HTML?

Correct linking in HTML head:

<link rel="stylesheet" href="styles.css">

Structure:

project-folder/
├── index.html
└── styles.css
<css>styles.css</css>
<link rel="stylesheet" href="styles.css">
<style src="styles.css">
<stylesheet>styles.css</stylesheet>
