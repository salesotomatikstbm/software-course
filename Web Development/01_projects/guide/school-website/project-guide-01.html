Step-by-Step: Build a School Webpage

Question: Take a look at the project page. What do you notice about how it's organized?
Hint: The page is divided into several sections. They are all arranged one below the other from top to bottom.
Hint: Instead of stacking them by default, the designer used flex properties like:
display: flex;
justify-content: center;
flex-direction: column;
align-items: center;
These make the layout neat and centered.

Question: Look at the header section of the page. What features can you spot there?
Hint:
  • The background color of the header is blue.
  • The school name appears in a large, bold font.
  • There's a tagline below the school name, in a smaller font.
  • You can also see some menu links.
  • There is extra space above and below the header contents, making it look clean and comfortable.
Hint: The code uses an h1 tag for the school name and a p tag for the tagline.
The menu links are placed inside a div with flex layout:
display: flex;
justify-content: center;
gap: 10px;
