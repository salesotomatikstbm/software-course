Introduction to CSS

Concept Description Projects Using This Concept
What is CSS? CSS (Cascading Style Sheets) is used to style HTML elements. All Projects
Ways to Apply CSS Three methods: Inline, Internal, and External CSS. All Projects
CSS Syntax Basic structure of CSS: selector { property: value; }. All Projects
Selectors Targets HTML elements for styling (e.g., element, class, ID, group selectors). All Projects
Inline CSS Applying styles directly within the style attribute of an HTML tag. Quick styling in small projects like Digital Birthday Card.
Internal CSS Defining styles inside a <style> tag within the HTML file. Small projects like Digital Birthday Card, To-Do List.
External CSS Linking an external CSS file to the HTML using <link> in <head>. Portfolio Website, Blog Page, E-Commerce Project.
CSS Comments (/* comment */) Used to document CSS code without affecting rendering. All Projects (for code maintainability).
Color Styling (color, background-color) Changes text and background colors using named colors, HEX, RGB, and HSL. Digital Birthday Card, Portfolio Website, Blog Page.
Text Styling (font-size, font-family, text-align) Adjusts text appearance like size, font type, and alignment. Portfolio Website, Blog Page, E-Commerce Project.
Class Selectors (.classname) Styles multiple elements with the same class name. All Projects (Reusable styles).
ID Selectors (#idname) Styles a unique element with an ID. Portfolio Website, E-Commerce Project (specific UI components).
Grouping Selectors (h1, h2, p {}) Apply styles to multiple elements at once. All Projects (for consistency).

CSS Selectors and Properties Breakdown

Concept Description Projects Using This Concept
Universal Selector (*) Applies styles to all elements on the page. Portfolio Website, Blog Page, E-Commerce Project.
Element Selector (p, h1, div, etc.) Targets all instances of a specific HTML element. All Projects.
Class Selector (.classname) Targets all elements with a specific class. All Projects (Reusable styles).
ID Selector (#idname) Targets a single unique element by ID. Portfolio Website, E-Commerce Project (specific UI components).
Grouping Selector (h1, h2, p {}) Applies the same styles to multiple elements. All Projects (for consistency).
Descendant Selector (div p) Targets an element inside another element. Blog Page (styling paragraphs inside a section).
Child Selector (div > p) Targets direct child elements only. Portfolio Website, E-Commerce Project.
Adjacent Sibling Selector (h1 + p) Targets an element immediately following another. Blog Page, Portfolio Website.
General Sibling Selector (h1 ~ p) Targets all siblings of a specified element. Blog Page, Portfolio Website.
Attribute Selector (input[type="text"]) Targets elements based on their attributes. To-Do List, E-Commerce Project (form inputs).
Pseudo-classes (:hover, :focus, :nth-child()) Targets elements in a specific state or position. All Projects (hover effects, button styles).
Pseudo-elements (::before, ::after) Inserts additional styling content before or after an element. Portfolio Website, Blog Page (decorative elements).

CSS Box Model Breakdown

Concept Description Projects Using This Concept
Box Model Overview Describes how elements are structured in terms of content, padding, border, and margin. All Projects (for layout and spacing).
Content Area The actual content inside an element (text, images, etc.). All Projects.
Padding (padding) Space between the content and the element’s border. Portfolio Website, Blog Page, E-Commerce Project (for spacing around text and images).
Border (border) The outline surrounding the element. Can be solid, dashed, or other styles. Digital Birthday Card, Portfolio Website, Calculator, E-Commerce Project.
Margin (margin) Space outside the element, separating it from other elements. All Projects (for layout spacing).
Width and Height (width, height) Defines the size of the element. Can be set in pixels, percentages, or auto. Portfolio Website, Blog Page, To-Do List, Calculator.
Box Sizing (box-sizing) Defines how the total size of an element is calculated (content-box vs. border-box). All Projects (to ensure consistent sizing).
Overflow (overflow) Controls how content is handled if it exceeds the element’s dimensions (hidden, scroll, auto). Blog Page (content overflow in sections), E-Commerce Project (product description area).
Max-width and Min-width (max-width, min-width) Defines the minimum and maximum width an element can have. Portfolio Website, Blog Page, E-Commerce Project (responsive design).

Colors, Backgrounds, and Borders Breakdown

Concept Description Projects Using This Concept
Color Property (color) Sets the text color using named colors, HEX, RGB, or HSL values. Portfolio Website, Blog Page, Digital Birthday Card, E-Commerce Project.
Background Color (background-color) Sets the background color of an element. All Projects (for section styling).
Background Image (background-image) Adds an image as the background of an element. Portfolio Website, Blog Page, Digital Birthday Card.
Background Size (background-size) Controls the size of the background image (cover, contain, etc.). Portfolio Website, Blog Page.
Background Repeat (background-repeat) Specifies if a background image should repeat (repeat, no-repeat, repeat-x, repeat-y). Digital Birthday Card, Portfolio Website.
Background Position (background-position) Defines the position of the background image (center, top, left, etc.). Portfolio Website, Blog Page.
Border (border) Defines the border around an element (width, style, color). Digital Birthday Card, Portfolio Website, Calculator, E-Commerce Project.
Border Radius (border-radius) Rounds the corners of an element. Digital Birthday Card, Calculator (button styling), E-Commerce Project (product cards).
Box Shadow (box-shadow) Adds shadow effects to elements. Portfolio Website, Blog Page, E-Commerce Project (for aesthetic improvements).
Opacity (opacity) Controls the transparency of an element (0 = fully transparent, 1 = fully visible). Portfolio Website (hover effects), Digital Birthday Card (fading effects).
Text Shadow (text-shadow) Adds shadow to text elements. Portfolio Website (name heading), Digital Birthday Card.
Gradient Backgrounds (linear-gradient, radial-gradient) Creates smooth color transitions. Portfolio Website, Digital Birthday Card, Blog Page.

Fonts and Text Styling Breakdown

Concept Description Projects Using This Concept
Font Family (font-family) Defines the typeface of text. Can be a web-safe font or a custom Google Font. Portfolio Website, Blog Page, Digital Birthday Card, E-Commerce Project.
Font Size (font-size) Adjusts the size of text. Can be in px, em, rem, %, or vw/vh. All Projects (headings, buttons, paragraphs).
Font Weight (font-weight) Sets the thickness of text (normal, bold, light, numeric values). Portfolio Website, Blog Page, To-Do List.
Font Style (font-style) Controls text appearance (normal, italic, oblique). Portfolio Website, Blog Page, Digital Birthday Card.
Text Color (color) Sets the color of text. All Projects (for better readability and aesthetics).
Text Alignment (text-align) Aligns text (left, right, center, justify). Blog Page, Digital Birthday Card, E-Commerce Project.
Text Decoration (text-decoration) Adds/removes underlines, overlines, or strikethroughs (underline, none, line-through). Blog Page (links), To-Do List (completed tasks).
Text Transform (text-transform) Changes text case (uppercase, lowercase, capitalize). Portfolio Website, Blog Page, E-Commerce Project.
Letter Spacing (letter-spacing) Adjusts space between characters. Portfolio Website (logo, headings).
Word Spacing (word-spacing) Adjusts space between words. Blog Page, Portfolio Website.
Line Height (line-height) Defines the space between lines of text. Blog Page (for readability), Portfolio Website.
Text Shadow (text-shadow) Adds shadow effects to text. Portfolio Website, Digital Birthday Card, Blog Page.

Positioning and Flexbox Breakdown

Concept Description Projects Using This Concept
Static Positioning (position: static) The default positioning of elements (normal document flow). All Projects (when no special positioning is needed).
Relative Positioning (position: relative) Positions an element relative to its normal position. Portfolio Website (for adjusting text/images slightly).
Absolute Positioning (position: absolute) Positions an element relative to its nearest positioned ancestor. Image Slider (for placing navigation arrows), Digital Clock.
Fixed Positioning (position: fixed) Fixes an element to the viewport, so it stays in place while scrolling. Portfolio Website (sticky navbar), To-Do List (floating button).
Sticky Positioning (position: sticky) A hybrid of relative and fixed positioning, where an element sticks to the viewport when scrolling past a certain point. Blog Page (sticky header), E-Commerce Project (sticky add-to-cart button).
Top, Left, Right, Bottom Properties Defines the positioning offsets for elements when position is set to relative, absolute, fixed, or sticky. Portfolio Website, Image Slider, Calculator.
Z-Index (z-index) Controls the stack order of overlapping elements. Image Slider (for layering images), Portfolio Website (overlapping text on images).
Display Flex (display: flex) Enables flexbox for a container, allowing child elements to be arranged efficiently. Portfolio Website, Blog Page, E-Commerce Project (product grid).
Flex Direction (flex-direction) Defines the direction of flex items (row, column, row-reverse, column-reverse). Portfolio Website, Blog Page (for responsive layouts).
Justify Content (justify-content) Aligns items along the main axis (flex-start, center, space-between, etc.). Portfolio Website, E-Commerce Project (product alignment).
Align Items (align-items) Aligns items along the cross-axis (flex-start, center, stretch). Portfolio Website (centering text), Digital Clock (aligning numbers).
Align Self (align-self) Aligns a single flex item within a flex container. Portfolio Website (aligning profile image differently from text).
Flex Wrap (flex-wrap) Allows items to wrap onto multiple lines if they exceed the container’s width. Blog Page, E-Commerce Project (product cards).
Flex Grow (flex-grow) Specifies how much an item should grow relative to others. Portfolio Website (for auto-resizing sections).
Flex Shrink (flex-shrink) Specifies how much an item should shrink relative to others. To-Do List (making buttons responsive).
Flex Basis (flex-basis) Defines the default size of a flex item before it grows/shrinks. Portfolio Website (controlling section width).
Gap (gap) Defines the space between flex items. Portfolio Website, Blog Page, E-Commerce Project (spacing between elements).

Basic Grid Layout Breakdown

Concept Description Projects Using This Concept
Display Grid (display: grid) Enables CSS Grid layout for an element, allowing precise placement of items in rows and columns. Portfolio Website, Blog Page, E-Commerce Project (product display).
Grid Template Columns (grid-template-columns) Defines the number of columns and their widths using values like px, %, fr, auto. Portfolio Website, Blog Page, To-Do List, E-Commerce Project.
Grid Template Rows (grid-template-rows) Defines the number of rows and their heights. Portfolio Website (hero section), Blog Page (content structure).
Grid Gap (gap or grid-gap) Specifies spacing between rows and columns. Portfolio Website, Blog Page, Digital Birthday Card.
Grid Auto Rows & Columns (grid-auto-rows, grid-auto-columns) Automatically assigns size to unspecified rows/columns. Blog Page (dynamic post listing), E-Commerce Project.
Justify Items (justify-items) Aligns grid items along the row axis (start, center, end, stretch). Portfolio Website, To-Do List, E-Commerce Project.
Align Items (align-items) Aligns grid items along the column axis (start, center, end, stretch). Portfolio Website, Blog Page.
Justify Content (justify-content) Controls the overall alignment of the grid itself inside the parent container. Portfolio Website (centering content).
Align Content (align-content) Aligns the entire grid’s content vertically. Portfolio Website (centering sections).
Grid Column Start & End (grid-column-start, grid-column-end) Defines where an item should start and end along columns. Portfolio Website (layout control), E-Commerce Project.
Grid Row Start & End (grid-row-start, grid-row-end) Defines where an item should start and end along rows. Portfolio Website, Blog Page.
Grid Area (grid-area) Assigns a name to a grid item for use in grid-template-areas. Portfolio Website, Blog Page.
Grid Template Areas (grid-template-areas) Defines layout structure using named areas for easy placement. Portfolio Website (header, main, sidebar, footer layout).
