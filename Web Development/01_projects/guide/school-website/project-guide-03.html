This section switches from regular images to icon packs. Using icons has its own standardized process.
For icons (such as Font Awesome), don’t forget to put their CDN link in the <head> of your HTML. For example:

Let’s look back and compile a checklist:
  • Learn how web pages fit together: header, content sections, and footer.
  • Use flexbox and grid for laying out content as needed.
  • Style everything using both Bootstrap and your own CSS.
  • Work with HTML tags: headings, paragraphs, links, images, tables, lists, etc.
  • Make sections pop with backgrounds, border effects, and spacing.
  • Set up navigation links with anchors and unique ids for sections.
  • Embed full pages or maps using iframe.
  • Add third-party icon sets (e.g., via Font Awesome).
  • Integrate fun extras like moving text or scrollers.
  • Frequently used CSS:
    display: flex;
    justify-content: center;
    align-items: center;
    flex-direction: column;
    gap: 10px;
    padding, margin, border-radius, background-color
  • Pay special attention to areas where you struggled and review them.
  • Recreate the layouts you practiced to reinforce what you learned.
